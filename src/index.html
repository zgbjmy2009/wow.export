Installing the wow.export Add-on for Blender 2.8

Blender users can make use of our special importer add-on which makes importing advanced objects as simple as a single click. WMO objects are imported with any exported doodad sets included. ADT map tiles are imported complete with all WMOs and doodads positioned as they would be in-game.

导出路径

导出文件的保存路径

启用共享纹理

如果启用，导出的纹理将导出到其自己的路径，而不是与其父级一起.

通过不复制纹理，大大减少了使用的磁盘空间.

使用绝对MTL路径

如果启用，则MTL文件将包含绝对纹理路径，而不是相对路径。

在计算机之间共享导出的模型时，这将导致问题。

如果在启用共享纹理的Cinema 4D中导入模型时遇到问题，请启用此选项.

CASC Locale

用于文件读取的语言环境。这只会影响游戏文件.

使用本地安装时，这应与客户端的语言环境匹配.

导出模型碰撞信息

如果启用，则导出为OBJ的M2模型还将其碰撞导出到.phys.obj文件中.

始终覆盖现有文件 (推荐的)

导出时，即使文件存在，也将始终将其写入磁盘。

禁用此功能可以加快导出速度，但是可能导致版本之间出现问题。

正则表达式过滤 (高级)

允许在过滤列表中使用正则表达式.

Copy File Directories

默认情况下，在wow.export中的文件列表上使用CTRL + C会将路径复制到剪贴板。

如果启用此选项，则仅复制所选路径的目录

Cache Expiry

闲置了几天后，缓存的数据将被删除。设置为零将禁用缓存清理(不推荐的).

手动清除缓存(需要重新启动)

尽管对高速缓存的内部整理大部分是自动的，但有时手动清除可以解决问题.

远程加密密钥地址

用于更新远程加密文件密钥的远程URL。（不建议）

添加加密密钥

手动添加BLTE加密密钥.

资源列表

用于更新CASC列表文件的远程URL或本地路径. (必须使用相同的格式)

列表文件更新频率

列表文件的更新频率（以天为单位）。设置为零以始终重新下载列表文件
